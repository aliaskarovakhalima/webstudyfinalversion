19. Website design

Change this:

<link rel="stylesheet" href="">

To this:

<link rel="stylesheet" href="style.css">

To use the stylesheet you must add a class to the HTML elements you want to style:

<div class="container black">

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

</div>

HTML / CSS Skeleton

<!DOCTYPE html>
<html lang="en">
<meta charset="UTF-8">
<title>Page Title</title>
<meta name="viewport" content="width=device-width,initial-scale=1">
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
<style>
</style>

</body>
</html>

Share your portfolio on GitHub


1. Register on https://github.com/, if you haven't done it yet.
2. Create a new repository named username.github.io , where username is your nickname on GitHub.
3. Create a new file. When the repository is not empty, there is a "Create New File" button in the upper right corner, but now, when there are no files yet, you can create the first one by direct link https://github.com/username/username .github.io/new/master , where username is your nickname.
4. Enter a name index.html and copy the contents of the file there index.html which you got in the previous lesson.
5. Click the "Commit new file" button.
6. Repeat the same for the style.css file.
7. Confirm the publication on GitHub Pages. In the repository settings: Settings -> Options Section -> GitHub Pages section.
8. In a few moments, the page will be available at your address https://username .github.io
